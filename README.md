🎮 Python Games Collection











This project showcases advanced Python development skills through two interactive games combining computer vision, real-time systems, and GUI design.

Built a gesture-controlled game using computer vision (MediaPipe + OpenCV), enabling real-time human interaction without traditional input devices
Designed a responsive game engine using Pygame with physics-based motion and collision handling
Developed a GUI-based word game using Tkinter demonstrating event-driven programming and user interaction
Implemented smooth animations, real-time tracking, and optimized rendering loops

📌 Projects Overview
🕹️ Hand-Controlled Pong

📄 File: pong_cv.py

A computer vision-powered Pong game where players control paddles using hand gestures via webcam.

🔥 Highlights
Real-time hand tracking (MediaPipe)
Smooth paddle control using interpolation
Physics-based ball movement
Multiplayer via dual-hand tracking
Live webcam overlay


🔤 Guess the Word Game

📄 File: pythongame.py

A GUI-based word guessing game demonstrating clean UI design and game logic.

🔥 Highlights
Interactive Tkinter UI
Input validation & feedback system
Lives-based gameplay
Event-driven architecture
⚙️ Installation
```
git clone https://github.com/S-Harshni/Python-Games.git
cd Python-Games
pip install pygame opencv-python mediapipe
```
▶️ Run
```
python pong_cv.py
python pythongame.py
```
